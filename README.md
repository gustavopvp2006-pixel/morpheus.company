# Morpheus Company 💎

A **Morpheus Company** é um projeto de uma loja virtual fictícia especializada em acessórios de correntes e pérolas. O site foi desenvolvido com foco em prática de desenvolvimento front-end, buscando um visual elegante, moderno e minimalista.

## 🚀 Sobre o projeto

Este projeto foi criado com o objetivo de praticar e aprimorar habilidades em desenvolvimento web, aplicando conceitos de estruturação de páginas, estilização e interatividade.

A proposta do site é simular uma loja virtual real, com uma identidade visual sofisticada voltada para acessórios de moda.

## 🛠️ Tecnologias utilizadas

- HTML5  
- CSS3  
- JavaScript  
- GitHub (versionamento e hospedagem)

## 🎯 Funcionalidades

- Página inicial com apresentação da marca  
- Catálogo de produtos de correntes e pérolas  
- Interface moderna e estilizada  
- Interações básicas com JavaScript  
- Layout organizado e responsivo  

## 📌 Objetivo do projeto

O principal objetivo foi colocar em prática conhecimentos de front-end, criando uma interface visual agradável e funcional, simulando uma loja online.
